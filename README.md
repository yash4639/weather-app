# 🌦️ Weather Dashboard (Streamlit + SQL Server)

## 📌 Project Overview

This project is a Weather Dashboard that fetches real-time weather data using an API and stores it in SQL Server. The data is displayed using an interactive Streamlit interface.

---

## 🛠️ Tech Stack

* Python
* Streamlit
* SQL Server
* OpenWeather API
* HTTPX & PyODBC

---

## ⚙️ Features

* 🌍 Select city from dropdown
* 🌡️ Fetch real-time weather data
* 🗄️ Store data in SQL Server
* 📊 Display weather metrics (temperature, humidity, etc.)
* 📈 Interactive dashboard UI

---

## 📂 Project Structure

* **app.py** → Streamlit application
* **requirements.txt** → Dependencies
* **images/** → Dashboard screenshots

---

## 🚀 How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Set API Key:

```bash
setx OPENWEATHER_API_KEY "your_api_key"
```

3. Run the app:

```bash
streamlit run app.py
```
